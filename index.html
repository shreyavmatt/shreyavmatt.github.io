Built a Web API for a Zoo Management System using Flask, visualised using Swagger with JSON.

The API allowed full management of a zoo, with the following features:

  • Add and remove animals from the zoo system
  • Register animal births
  • Monitor and register animal enclosures
  • Create animal feeding, medical care and enclosure cleaning schedules
  • Add and remove employees from the zoo system
  • View and edit employee stats (how many animals and enclosures they are responsible for, etc.)
  • Track when scheduled tasks are completed (feeding, cleaning, medical care)
